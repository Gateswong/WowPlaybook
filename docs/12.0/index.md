# 至暗之夜第1赛季总览

## 可玩内容

- 11个地下堡、8个副本、3个团本、8个大秘境。
- 强化符文石：永歌森林，每周一次。
- 丰饶：每周每角色获取<c=3376>（最多8枚），可通过丰饶活动（在4地图轮换）换取<c=3377>，可获得专业增益和一些收藏内容。
- 世界BOSS：四个地图中的某一个地图会有世界BOSS。整个战网每周第一次击杀会奖励战团绑定的勇士装备。
- 狩猎：银月城旅馆下层接取，每周每角色4次梦魇狩猎。前两次梦魇狩猎奖励勇士装备。周常要求3次梦魇狩猎，给地下堡招大BOSS道具（藏宝图相关）。
- 斯托马兰突袭战：虚影风暴的物料世界事件，塔防。可以挂机练小号。
- 屠戮者高地：每4个小时刷新一个可PVP可PVE做的世界任务。还有相关PVP玩法。
- 虚空突袭：又一个无聊的这打打那打打最后去一个地方打打打打的活动。奖励代币换T2换色外观，或是500代币换英雄装备（不如地下堡一根）。
- ~~地上堡~~ 干扰仪式：小型地下城刷代币换T2换色外观或是英雄装备（真到轻松刷的时候还真需要英雄装备吗？这代币还不能角色间转移）。
- Roll币：一周俩，可以roll两次大秘境或一次团本装备（真有人会在地下堡roll吗？）
- 升装备道具：12.0.5开启4周后可获取的升级武器/饰品装等的道具。
- ~~潜水员戴夫~~ 深渊垂钓：奖励幻化小宠物，纯娱乐玩法（别说还挺好玩的其实）。
- 草药学护土：每小时可以使用10个<i=236761>制作一个护土，可以保证摘取一枚<i=236780>。
- 威风野兽剥皮：剥皮师每天可以在4个不同地点召唤威风野兽并剥皮，可以获取一些稀有材料。
