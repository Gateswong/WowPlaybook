# 平衡德鲁伊

## 准备工作

<CopyBox title="鸟德的冷却管理器配置" file="BalanceDruidCDM" />

配装：急速450-600后全给暴击喝精通；暴击+165后略微大于精通；全能0，最多戴一个套装手。喝金星暴击合剂（<i=241326>）后暴击绿字略微高于精通绿字，急速大约10%出头即可。鲁莽药水会提高暴击，然后开蓝眼睛增加大量精通（参考下面爆发宏）。

附魔选择：头肩鞋选一个喜欢的第3属性。极致输出选加速，极致生存选闪避，兼顾奶妈选吸血。大宝石选择32主属性宝石。

爆发宏：按优先级使用<i=245902>（金星）、<i=245898>（金星）和<i=241288>、<i=241308>的金银星版本（优先金星）。

主动饰品：<i=249346>、<i=249343>。
