# 制造业

## 前期知识点来源

- 地图拾取知识点道具。数量未知，每个3点知识点。（地心之战是8个共24点知识点）
- 声望6级可以买知识点书，10点，需要75<c=3316>和专业专属的匠人代币（例如<c=3256>）。
  - 银月城：<i=257601,,!裁缝>、<i=257600,,!附魔>、<i=257599,,!珠宝加工>。
  - 阿曼尼巨魔：采矿<i=250924>、剥皮<i=250923>、制皮<i=250922>。
- 制造业周常任务：给3点知识点。例如：<q=93714>。
- 铭文论述：每周给1点。例如：<s=1230034>。


## 版本初期冲制造业的方案

- 准备一个制造专业装备的小号，将专业装备点到全身绿色制造业装备可以保证紫色装备5星为止。后续这个小号可以点一些零碎的点。
- 铭文（专业装备小号）知识点：Blueprints(30)、Market Research(25)。制造铭文工具、炼金工具、烹饪工具。
- 锻造（专业装备小号）知识点：Crastsmithing(25)、Trade Tools(20)、Trade Accessories(10)。制造制皮工具、剥皮工具、锻造工具、草药学工具、采矿工具，锻造配件、制皮配件、裁缝配件。
- 裁缝（专业装备小号）知识点：Sin'dorei Finery(20)、Fiber Arts(15)。制造裁缝衣服（黄布）、炼金衣服（紫布）、厨师帽子（黄布）、附魔帽子（紫布）、草药学帽子（紫布）。
- 裁缝（搓布小号）知识点：Fiber Arts(15)、Creative Efficiency(10)、Textile Utilization(10)、Nimble Needlework(5)。制造黄布或者紫布选一个。3天4张。
- 制皮（专业装备小号）知识点：Flawless Fortes(10)、Artisanal Accessories(30)。制造炼金帽子、珠宝衣服、草药学披风、制皮衣服、锻造衣服、工程手套、剥皮帽子、剥皮披风。
- 制皮（皮甲/锁甲号）知识点：对应专精中间(20)+两边(各30)点满，共80点，可解锁所有紫色装备。
