# 通用宏命令

## 坐骑宏

```
#showtooltip
/use [mod:altshift] Sandstone Drake; [mod:ctrlshift] Switch Flight Style; [nomod,btn:2] Trader's Gilded Brutosaur; [mod:alt] Ashes of Al'ar; [nomod] Glacial Tidestorm; [mod:ctrl] Winterspring Frostsaber; [mod:shift] Grand Expedition Yak
```

## 传送宏

```
#showtooltip
/cast [mod:ctrlshift] Flight Master's Whistle; [mod:shift] Dalaran Hearthstone; [mod:alt][nomod,btn:2] Personal Key to the Arcantina; [mod:ctrl] Teleport to Plot; [mod:ctrlalt] Garrison Hearthstone; [nomod,btn:1] Lunar Elder's Hearthstone
```
