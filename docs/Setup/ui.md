## 调整动作条样式

- 安装插件Masque，并安装如下皮肤：Hex、Cooldown Manager、Zurrn、Blizzard Bars。
- 在Masque设置中设置暴雪动作条：
  - 动作条2：Cooldown Manager
  - 动作条8：Zurrn (Circle)
  - 其余动作条：Hex Black Border Soft
